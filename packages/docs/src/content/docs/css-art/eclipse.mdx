# Eclipse

A pure CSS solar eclipse illustration with a moon silhouette against a layered corona glow.

## Installation

```bash
npm install @duskmoon-dev/el-art-eclipse
```

## Usage

```javascript
import '@duskmoon-dev/el-art-eclipse/register';
```

```html
<el-dm-art-eclipse></el-dm-art-eclipse>
```
