# Nested Menu

A sidebar navigation element with collapsible menu levels for building hierarchical navigation structures. Renders as a `<ul>` list with slotted content.

## Installation

```bash
npm install @duskmoon-dev/el-nested-menu
```

## Usage

```javascript
import { register } from '@duskmoon-dev/el-nested-menu';
register();
```

```html
<el-dm-nested-menu>
  <li><a href="/docs">Getting Started</a></li>
  <li>
    <details>
      <summary>Components</summary>
      <ul>
        <li><a href="/docs/button">Button</a></li>
        <li><a href="/docs/input">Input</a></li>
      </ul>
    </details>
  </li>
</el-dm-nested-menu>
```

## Slots

| Slot | Description |
|------|-------------|
| Default | Menu items (`<li>` elements with links, details/summary for collapsible sections, or `.nested-menu-title` for section headers) |

## CSS Classes

| Class | Description |
|-------|-------------|
| `nested-menu-title` | Apply to `<li>` elements to render as a non-clickable section header |
