# Navigation

A composite navigation layout container element for building navigation structures. Acts as a semantic wrapper for navigation items.

## Installation

```bash
npm install @duskmoon-dev/el-navigation
```

## Usage

```javascript
import { register } from '@duskmoon-dev/el-navigation';
register();
```

```html
<el-dm-navigation>
  <a href="/home">Home</a>
  <a href="/about">About</a>
  <a href="/contact">Contact</a>
</el-dm-navigation>
```
