# Multi Select

A dropdown element for selecting multiple items from a list. Selected items are displayed as chips with a customizable placeholder and color theming.

## Installation

```bash
npm install @duskmoon-dev/el-multi-select
```

## Usage

```javascript
import { register } from '@duskmoon-dev/el-multi-select';
register();
```

```html
<el-dm-multi-select placeholder="Choose tags...">
  <!-- options -->
</el-dm-multi-select>
```

## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `placeholder` | `string` | `'Select...'` | Placeholder text when nothing is selected |
| `disabled` | `boolean` | `false` | Disable the select |
| `color` | `'primary' \| 'secondary' \| 'tertiary' \| 'success' \| 'warning' \| 'error'` | `'primary'` | Color theme |

## Slots

| Slot | Description |
|------|-------------|
| Default | Dropdown options/content |
