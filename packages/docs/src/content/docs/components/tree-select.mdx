# Tree Select

A hierarchical dropdown selection component for presenting tree-structured data. Renders a trigger button with a placeholder and a slot for tree-structured options.

## Installation

```bash
npm install @duskmoon-dev/el-tree-select
```

## Usage

```javascript
import { register } from '@duskmoon-dev/el-tree-select';
register();
```

```html
<el-dm-tree-select placeholder="Select department...">
  <!-- tree options -->
</el-dm-tree-select>
```

## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `placeholder` | `string` | `'Select...'` | Placeholder text when nothing is selected |
| `disabled` | `boolean` | `false` | Disable the select |
| `color` | `'primary' \| 'secondary' \| 'tertiary' \| 'success' \| 'warning' \| 'error'` | `'primary'` | Color theme |

## Slots

| Slot | Description |
|------|-------------|
| Default | Tree-structured items/options |
