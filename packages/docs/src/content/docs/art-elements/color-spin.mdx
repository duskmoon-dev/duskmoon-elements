# Color Spin

A pure CSS animated color wheel with four spinning segments that create a mesmerizing rotation effect.

## Installation

```bash
npm install @duskmoon-dev/el-art-color-spin
```

## Usage

```javascript
import '@duskmoon-dev/el-art-color-spin/register';
```

```html
<el-dm-art-color-spin></el-dm-art-color-spin>
```
