# Synthwave Starfield

A pure CSS retro synthwave scene with a perspective grid, animated stars, and neon gradient sky — a nostalgic 80s aesthetic.

## Installation

```bash
npm install @duskmoon-dev/el-art-synthwave-starfield
```

## Usage

```javascript
import '@duskmoon-dev/el-art-synthwave-starfield/register';
```

```html
<el-dm-art-synthwave-starfield></el-dm-art-synthwave-starfield>
```
