# CSS Switch

A pure CSS Nintendo Switch console illustration with detailed joy-cons, buttons, and screen.

## Installation

```bash
npm install @duskmoon-dev/el-art-csswitch
```

## Usage

```javascript
import '@duskmoon-dev/el-art-csswitch/register';
```

```html
<el-dm-art-csswitch></el-dm-art-csswitch>
```

## Live Demo

</ComponentDemo>

## CSS Custom Properties

| Property | Default | Description |
|----------|---------|-------------|
| `--art-csswitch-size` | `800px` | Width of the console |
| `--joycon-left` | `#00c5e2` | Left joy-con color |
| `--joycon-right` | `#ff6658` | Right joy-con color |
