# Atom

A pure CSS atom illustration featuring a nucleus with animated orbiting electrons.

## Installation

```bash
npm install @duskmoon-dev/el-art-atom
```

## Usage

```javascript
import '@duskmoon-dev/el-art-atom/register';
```

```html
<el-dm-art-atom></el-dm-art-atom>
```

## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of the atom |
| `glow` | `boolean` | `false` | Adds a glow effect to the nucleus |
