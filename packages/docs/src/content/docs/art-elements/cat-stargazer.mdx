# Cat Stargazer

A pure CSS illustration of a cat sitting on a hill, gazing up at the moon on a starry night.

## Installation

```bash
npm install @duskmoon-dev/el-art-cat-stargazer
```

## Usage

```javascript
import '@duskmoon-dev/el-art-cat-stargazer/register';
```

```html
<el-dm-art-cat-stargazer></el-dm-art-cat-stargazer>
```
