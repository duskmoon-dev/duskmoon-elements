# Mountain

A pure CSS aurora borealis night scene with layered mountain silhouettes, pine trees, and animated northern lights.

## Installation

```bash
npm install @duskmoon-dev/el-art-mountain
```

## Usage

```javascript
import '@duskmoon-dev/el-art-mountain/register';
```

```html
<el-dm-art-mountain></el-dm-art-mountain>
```

## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of the scene |

## CSS Custom Properties

| Property | Default | Description |
|----------|---------|-------------|
| `--art-mountain-size` | `600px` | Width and height of the canvas |
