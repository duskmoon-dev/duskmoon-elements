# Mountain

A pure CSS mountain landscape with single peak and mountain range variants, plus optional sunset mode.

## Installation

```bash
npm install @duskmoon-dev/el-art-mountain
```

## Usage

```javascript
import '@duskmoon-dev/el-art-mountain/register';
```

```html
<el-dm-art-mountain></el-dm-art-mountain>
```

## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `'single' \| 'range'` | `'single'` | Single peak or mountain range |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size of the illustration |
| `sunset` | `boolean` | `false` | Applies warm sunset color palette |
